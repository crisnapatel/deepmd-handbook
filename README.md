# DeePMD + dpgen — But Make It Make Sense

A conversational tutorial for the DeePMD-kit + dpgen active learning pipeline.

Built with [Jupyter Book](https://jupyterbook.org/).

## Building locally

```bash
pip install -r requirements.txt
jupyter-book build .
```

Open `_build/html/index.html` in your browser.
